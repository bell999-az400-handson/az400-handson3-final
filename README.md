# az400-handson3-final

This repository is a hands-on project for **AZ-400: Designing and Implementing Microsoft DevOps Solutions**
aligned with the **latest 2026 exam syllabus**.

## Goals
- Design and implement CI/CD using Azure DevOps
- Use GitHub and Azure DevOps together
- Apply security, IaC, and monitoring
- Explain *why* a design was chosen (AZ-400 key skill)

## Scope
- Azure Boards
- Azure Repos / GitHub
- Azure Pipelines (YAML)
- Infrastructure as Code (Bicep)
- Azure Key Vault
- Application Insights
- GitHub Copilot assisted development

## Expected Outcome
- Secure multi-stage YAML pipeline
- Trunk-based development with PR policies
- CI + CD with approvals
- Monitoring and feedback loop
